# MahoganyQen

Projekt React + backend Node.js dla strony MahoganyQen.

## Struktura projektu

- `my-app/` – frontend React
- `backend/` – serwer Node.js / Express
- `public/` – zasoby publiczne
- `src/` – komponenty React i logika aplikacji

## Jak uruchomić projekt

1. Zainstaluj zależności frontendu:

```bash
cd my-app
npm install
cd backend
npm install

# uruchom backend
node server.js
# uruchom front end
cd my-app
npm start
