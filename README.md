# MahoganyQen - Full-Stack Web Application

A modern web application built with a focus on seamless user experience and robust server-side logic. This project demonstrates the integration of a **React** frontend with a **Node.js/Express** backend.

## 🚀 Features

* **Dynamic Frontend:** Built with React for a fast, responsive UI.
* **Custom Backend:** Node.js server handling API requests and server-side logic.
* **Modern Styling:** Utilizes SASS and CSS3 for high-quality visuals.
* **Scalable Architecture:** Clear separation between frontend and backend logic.

## 🛠 Tech Stack

* **Frontend:** React.js, JavaScript (ES6+), SASS, HTML5, CSS3.
* **Backend:** Node.js, Express.
* **Tools:** npm, Git.

---

## 📂 Project Structure

* `my-app/` – Frontend React application.
* `backend/` – Node.js / Express server.
* `public/` – Static assets and public resources.
* `src/` – React components and core application logic.
